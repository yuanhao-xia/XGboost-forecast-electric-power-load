# XGboost-forecast-electric-power-load
使用XGboost进行电力负荷数值的未来预测，训练24个单点XGboost模型输未来24个点的预测值，基于CNN-LSTM预测代码的修改而来，改变了模型和训练参数的设置，实测的预测效果遇到了些问题，模型对极端的变化值预测并不准确，可能是训练方式的问题，但其他情况的拟合情况是非常好的。

训练过程如下图，最后的RMSE趋向0.03稳定，但仍然存在肉眼可见的预测缺陷，对于极端值的预测不稳定而且会低估，不太理解为什么会出现这种情况，现在的结论（大概不确定）是xgboost模型分为了24个，在数值突变的点预测结果会趋向于保守，因为预测数值大的情况下归一化后的误差计算会放大，预测值小一点的话归一化误差不会有太大影响（7点负荷急剧突变，很难预测准确）
